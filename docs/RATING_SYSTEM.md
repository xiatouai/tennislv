# Rating System — TennisLV

## 概述

TennisLV 评级体系参考 NTRP（National Tennis Rating Program）能力描述，为业余网球选手生成 **1.0-7.0**（每 0.5 一级）的 AI 预估网球评级。

**重要：** 本系统不等同于 USTA 官方 NTRP 评级。

## 评级类型（分层体系）

| 类型 | 标识 | 置信度范围 | 数据来源 |
|------|------|-----------|---------|
| 问卷预估 | `questionnaire_estimate` | 60-70% | 8 题问卷加权评分 |
| 视频增强 | `video_enhanced` | 80-85% | 问卷 + 视频动作分析 |
| 球友认证 | `peer_verified` | 75-85% | 问卷/视频 + 球友投票 |
| 教练认证 | `coach_verified` | 90-95% | 教练人工评定 |

**规则：**
- 默认完成问卷后生成"问卷预估"，不调用大模型
- 视频上传不等于自动分析——用户需主动点击"开始视频增强评级"
- 球友认证需≥2 票后升级评级类型
- 教练认证优先级最高

## NTRP 风格等级定义

### 1.0-2.5：初学到入门

| 等级 | 名称 | 核心特征 |
|------|------|---------|
| 1.0 | 初学入门 | 刚开始学，正在学习规则、握拍和基本挥拍 |
| 1.5 | 基础练习 | 能偶尔把球打过网，但稳定性差 |
| 2.0 | 基础建立期 | 能基本对打，但节奏、选球和稳定性不足 |
| 2.5 | 初级进阶级 | 能维持短回合，开始控制方向 |

### 3.0-4.0：中级到进阶

| 等级 | 名称 | 核心特征 |
|------|------|---------|
| 3.0 | 中级入门 | 稳定底线回合，具备基本发球和比赛能力 |
| 3.5 | 中级稳定 | 击球有稳定性和方向控制，开始使用战术 |
| 4.0 | 中高级 | 技术全面，稳定打出战术型网球 |

### 4.5-5.5：高级到准职业

| 等级 | 名称 | 核心特征 |
|------|------|---------|
| 4.5 | 高级入门 | 技术全面，能在业余比赛中形成压制力 |
| 5.0 | 高级 | 接近大学校队水平，能用稳定武器赢球 |
| 5.5 | 准专业级 | 具备准职业水准，可在高水平赛事中竞争 |

### 6.0-7.0：职业级

| 等级 | 名称 | 核心特征 |
|------|------|---------|
| 6.0 | 专业级 | 拥有职业选手经验 |
| 6.5 | 专业高级 | 拥有职业积分或排名 |
| 7.0 | 精英级 | 世界级职业选手 |

完整能力描述见 `packages/shared/src/ntrp.ts` 中的 `NTRP_LEVELS`。

## 评分算法

### 问卷预估评分（当前实现）

**输入：** 8 道问卷答案

**维度加权：**

| 维度 | 对应问题 | 权重 |
|------|---------|------|
| 底线相持能力 | Q3 | 25% |
| 发球稳定性 | Q4 | 20% |
| 网前截击能力 | Q5 | 10% |
| 移动与防守 | Q6 | 15% |
| 比赛经验 | Q7 | 20% |
| 自评等级 | Q8 | 10% |

**步骤：**

1. 每题答案映射为 1-4 分
2. 加权求和得加权平均分 (1.0-4.0)
3. 映射到 NTRP 风格等级 (1.0-7.0)

```
加权平均 ≤ 1.15 → 1.0
加权平均 ≤ 1.35 → 1.5
加权平均 ≤ 1.60 → 2.0
加权平均 ≤ 1.85 → 2.5
加权平均 ≤ 2.15 → 3.0
加权平均 ≤ 2.45 → 3.5
加权平均 ≤ 2.75 → 4.0
加权平均 ≤ 3.05 → 4.5
加权平均 ≤ 3.35 → 5.0
加权平均 ≤ 3.65 → 5.5
加权平均 ≤ 3.90 → 6.0
> 3.90          → 6.5
```

4. 球龄(Q1)和打球频率(Q2)用于微调置信度，不直接参与等级计算

### 视频增强评分（设计中）

当用户上传视频并触发视频增强时：
- 问卷结果权重 60%
- 视频动作稳定性 25%
- 视频中移动和击球质量 15%

### 球友认证评分（设计中）

- 问卷/视频结果权重 70%
- 球友认证 30%（≥2 票生效）
- 打过球的球友认证权重高于未打过

### 教练认证（设计中）

- 教练给出的等级优先级最高
- 覆盖其他评级来源

## 置信度计算

| 评级类型 | 基础置信度 | 调节因素 |
|---------|-----------|---------|
| 问卷预估 | 60% | + (球龄分 + 频率分) / 2 × 2 |
| 视频增强 | 82% | + 同上 |
| 球友认证 | 78% | + 认证人数 × 2 |
| 教练认证 | 92% | 固定 |

置信度标签：
- ≥ 90% → 高
- ≥ 80% → 较高
- ≥ 65% → 中
- < 65% → 低

## 自评对比

系统将用户 Q8 自评等级与系统预估等级进行对比：

- 差异 ≤ 0.25：自评与系统预估基本一致
- 自评 > 系统：提示差异可能来自发球稳定性和比赛经验维度，建议上传视频
- 自评 < 系统：实际能力可能高于自评，建议邀请球友认证验证

## 适合约球范围

基于评级等级 ±0.5：
- 例如 3.0 → 2.5-3.5
- 例如 4.5 → 4.0-5.0

范围用于社交匹配和约球局等级要求。

## 维度评分展示

每个维度输出 0-100 分，附带文本标签：

| 分数区间 | 标签 |
|---------|------|
| ≥ 85 | 较强 |
| ≥ 70 | 中级 |
| ≥ 50 | 初级 |
| < 50 | 入门 |

展示格式：`75/100 中级`
