# PRD — TennisLV：业余网球评级与同城约球平台

## 产品定位

TennisLV 是一个面向业余网球爱好者的 **NTRP 风格评级 + 同城约球** 平台。

核心价值：
- 通过问卷和视频生成 AI 预估网球等级（参考 NTRP 能力描述）
- 等级用于约球、组局、俱乐部分级和球友认证
- 连接同城同水平球友，降低约球匹配门槛

**重要措辞：**
- 使用"参考 NTRP 能力描述""NTRP 风格等级"
- 不使用"官方 NTRP 评级""USTA 官方认证"
- 结果统一称为"AI 预估评级""业余网球评级"

## 产品形态

| 阶段 | 形态 | 用途 |
|------|------|------|
| Phase 1 | H5 网页 | 获客、传播、分享 |
| Phase 2-5 | 微信小程序 | 账户、等级档案、同城球友、约球 |

当前处于 Phase 1（H5 评级传播），Phase 2-5 为后续规划。

## 核心产品闭环

```
[完成 NTRP 风格问卷] → [生成业余网球评级]
        ↓
[可选上传视频] → [视频增强评级]
        ↓
[微信登录] → [保存评级到个人资料]
        ↓
[等级展示] → [同城同水平球友匹配]
        ↓
[发布约球局] → [其他球友报名]
        ↓
[赛后评价] → [球友认证] → [评级置信度提升]
```

## 功能总览

### Phase 1：H5 评级传播（当前阶段，已完成）

| 功能 | 描述 | 状态 |
|------|------|------|
| NTRP 风格问卷 | 8 道题，对应 NTRP 能力维度 | ✅ 已完成 |
| 问卷预估评级 | 加权评分算法生成 1.0-7.0 评级 | ✅ 已完成 |
| 视频上传 | 可选上传打球视频 | ✅ mock |
| 视频增强评级 | 点击后进入视频分析流程获取更高置信度 | ✅ mock |
| 评级结果页 | 等级、名称、特征、维度评分、自评对比 | ✅ 已完成 |
| 分层评级体系 | 问卷预估/视频增强/球友认证/教练认证 | ✅ 已完成 |
| 分享评级卡 | 含 NTRP 风格等级、典型特征、QR 码 | ✅ 已完成 |
| 分享海报 | 可截图保存的海报 | ✅ 已完成 |
| 球友认证 | 邀请球友评价水平（偏低/差不多/偏高） | ✅ 已完成 |
| URL 分享 | hash-based 分享链接 | ✅ 已完成 |

### Phase 2：小程序账户与等级档案

| 功能 | 描述 |
|------|------|
| 微信登录 | wx.login + 手机号授权 |
| 个人资料 | 头像、昵称、城市、常打球场、打球频率 |
| 等级档案 | 展示当前 NTRP 风格等级 + 评级类型 |
| 评级历史 | 多次评级记录、等级变化趋势 |
| 球友认证记录 | 收到的认证一览 |
| 等级标识分享 | 小程序码 + 等级卡片 |

### Phase 3：同城球友

| 功能 | 描述 |
|------|------|
| 城市选择 | 当前城市 + 可选其他城市 |
| 常打球场 | 从球场列表选择或自定义添加 |
| 等级筛选 | 按 NTRP 风格等级范围筛选球友 |
| 同水平推荐 | 根据等级匹配推荐球友 |
| 申请加球友 | 发送申请，对方确认后成为球友 |

### Phase 4：同城约球

| 功能 | 描述 |
|------|------|
| 发布球局 | 时间、球场、人数、等级要求 |
| 球局要求 | 设置最低/最高等级（如 2.5-3.5） |
| 报名与审核 | 球友报名，发起人确认或拒绝 |
| 球局分享 | 分享球局到微信群 |
| 赛后评价 | 球友互评、水平反馈 |

### Phase 5：评级校准

| 功能 | 描述 |
|------|------|
| 球友认证权重 | 打过球的球友认证权重高于未打过 |
| 教练认证 | 教练账号可认证选手等级 |
| 对战反馈 | 赛后水平反馈纳入评级调整 |
| 比赛结果 | 正式比赛结果作为评级参考 |
| 动态置信度 | 多种反馈源综合调整评级置信度 |

## 评级体系

### 分层评级类型

| 类型 | 置信度 | 说明 |
|------|--------|------|
| 问卷预估 | 60-70% | 基于 8 题问卷的加权评分 |
| 视频增强 | 80-85% | 问卷 + 视频分析 |
| 球友认证 | 75-85% | 问卷/视频 + 球友投票 |
| 教练认证 | 90-95% | 教练人工评定 |

### NTRP 风格等级 (1.0-7.0)

| 等级 | 名称 | 典型特征 |
|------|------|---------|
| 1.0 | 初学入门 | 刚开始接触网球 |
| 1.5 | 基础练习 | 能偶尔把球打过网 |
| 2.0 | 基础建立期 | 能进行基本对打 |
| 2.5 | 初级进阶级 | 能维持短回合 |
| 3.0 | 中级入门 | 稳定底线回合，具备基本比赛能力 |
| 3.5 | 中级稳定 | 技术基本成熟，多拍能力强 |
| 4.0 | 中高级 | 技术全面，有得分手段 |
| 4.5 | 高级入门 | 能形成压制力 |
| 5.0 | 高级 | 接近高水平业余水平 |
| 5.5 | 准专业级 | 具备准职业水准 |
| 6.0 | 专业级 | 拥有职业选手经验 |
| 6.5 | 专业高级 | 拥有职业积分或排名 |
| 7.0 | 精英级 | 世界级职业选手 |

## 用户流程

### H5 评级流程（当前）
```
[首页] → [开始评级]
  → [8 道问卷] → [完成问卷]
  → [视频页：上传/跳过] → [生成评级]
  → [结果页] → [生成评级卡] → [分享海报]
            → [邀请球友认证]
```

### 小程序完整流程（后续）
```
[微信登录] → [个人资料]
  → [评级 Tab]：完成问卷 → 保存评级 → 查看等级档案
  → [约球 Tab]：发布球局 / 浏览球局 / 报名
  → [球友 Tab]：同城球友 / 等级筛选 / 加球友
  → [我的]：等级档案 / 评级历史 / 认证记录 / 设置
```

## 小程序底部 Tab

| Tab | 内容 |
|-----|------|
